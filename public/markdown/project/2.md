### Architecture

![AWS 전체 아키텍처 구성도](/images/project/2-aws-arch.png)

AWS VPC 내 Public / Private Subnet 구조로 인프라를 구성하고 EKS 기반 마이크로서비스를 Private Subnet에서 운영했습니다.
CloudFront + S3를 통해 정적 리소스를 제공하고 NLB + Istio Gateway를 통해 Kubernetes 서비스로 트래픽을 전달하도록 설계했습니다.
데이터 계층은 RDS Cluster와 Redis Cluster로 구성했습니다.

---

### Tech Stack

- **Infrastructure** : AWS (EKS, RDS, ElastiCache, S3, CloudFront, ECR, IAM, Secrets Manager), Terraform, Atlantis
- **Container & Platform** : Docker, Kubernetes(EKS), Helm, Istio
- **CI/CD** : GitHub Actions, ArgoCD
- **Observability** : Prometheus, Grafana, Loki, Jaeger, Kiali
- **Security** : IRSA, External Secrets Operator, Kubernetes Secret / ConfigMap

---

### Role

* Dockerfile 작성 및 애플리케이션 컨테이너 이미지 빌드
* Kubernetes Manifest 직접 작성 및 마이크로서비스 배포 구성
* Helm chart 설계 및 템플릿 기반 배포 구조 구현
* External Secrets Operator(ESO) + IRSA 기반 시크릿 관리 구조 구현
* EKS 환경에서 MSA 애플리케이션 배포 및 운영
---

### MSA Architecture (EKS)

![EKS 아키텍처 구조](/images/project/2-eks.png)

AWS EKS 환경에서 Auth, Flight, Booking, Payment, Admin의 5개 마이크로서비스를 배포했습니다.
Deployment, Service, ConfigMap, Secret, ServiceAccount 리소스를 직접 작성하여 서비스 단위 독립 배포가 가능하도록 구성했습니다.

Helm chart를 작성하여 Kubernetes 리소스를 템플릿화하고 values.yaml을 통해 이미지 버전, replica 수, 환경 변수 등을 관리하도록 구성했습니다.

---

### Service Mesh (Istio)

Istio Service Mesh를 적용하여 서비스 간 통신을 제어하고 관찰 가능성을 확보했습니다.
Istio IngressGateway와 AWS Network Load Balancer를 연동하여 외부 트래픽 진입점을 단일화하고 서비스 간 트래픽 제어 구조를 구성했습니다.

---

### Secret Management (ESO + IRSA)

![External Secrets Operator 동작 구조](/images/project/2-eso.jpg)

AWS Secrets Manager와 External Secrets Operator를 연동하여 Kubernetes Secret을 자동 동기화하는 구조를 구현했습니다.
IRSA를 통해 Pod 단위 IAM 권한을 부여하여 민감정보 접근을 제어하고 Kubernetes Secret을 Git 저장소에 직접 저장하지 않도록 구성했습니다.

---

### CI/CD (GitOps)

GitHub Actions를 통해 애플리케이션 빌드와 Docker 이미지 생성을 자동화하고 Amazon ECR에 푸시하도록 구성했습니다.
ArgoCD 기반 GitOps 배포 구조를 구성하여 Kubernetes Manifest 변경 시 자동으로 EKS 클러스터에 반영되도록 구현했습니다.

---

### Observability

Prometheus + Grafana를 통해 Kubernetes 자원 사용량과 서비스 상태를 모니터링하고 Loki를 통해 로그를 수집했습니다.
Jaeger를 활용하여 분산 트랜잭션 추적을 구성했으며 Kiali를 통해 Service Mesh 트래픽 흐름을 시각화했습니다.
