- **Front-end 개발**
  - 사용자 인증 및 관리: 회원가입, 로그인 페이지 및 유효성 검사 로직 구현. Redux를 활용한 사용자 인증 상태 관리
  - 마이페이지 및 대시보드: 사용자 프로필 관리 및 개인화된 산책 기록 리스트 인터페이스 설계
  - 실시간 산책 타이머: 산책 시작/종료 시간을 관리하고 실시간으로 경과 시간을 표시하는 타이머 기능 구현

- **배포 및 운영 자동화**
  - CI/CD 파이프라인 구축: GitHub Actions를 사용하여 코드 Push 시 자동으로 빌드 및 배포가 진행되도록 자동화 프로세스 구축
  - AWS 기반 정적 호스팅: AWS S3와 CloudFront를 결합하여 보안성이 강화된(HTTPS) 고성능 콘텐츠 전송 환경 구축
  - 도메인 관리: Route53을 연동하여 커스텀 도메인 환경 최적화

- **성과 및 핵심 역량**
  - 배포 자동화: 수동 배포 과정을 CI/CD로 대체하여 개발 생산성 향상 및 배포 오류 최소화
  - 인프라 최적화: CloudFront(CDN) 도입을 통해 전역 서비스 속도를 개선하고 S3 오리진 보안 강화
